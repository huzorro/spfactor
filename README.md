# spfactor

## Usage of spfactor
~~~
  -cache=false: cache start
  -db=10: max db open connections
  -finalStat=false: final stat start
  -moHandler=false: mo handler start
  -moReceiver=false: mo receiver start
  -moRest=false: mo rest start
  -mtHandler=false: mt handler start
  -mtReceiver=false: mt recevier start
  -mtRest=false: mt rest start
  -onlyMtHandler=false: only handler mt start
  -onlyMtReceiver=false: only receiver mt
  -port=":10086": service port
  -redis=20: redis idle connections
~~~
